## Общие сведения

**Конфигуратор 1С** — основной инструмент разработки на платформе 1С:Предприятие. Это встроенная среда, в которой выполняется большая часть работы: редактирование метаданных, написание кода, отладка, работа с хранилищем конфигурации.

Конфигуратор остаётся центральным инструментом, но для повышения качества кода и удобства работы используются дополнительные инструменты.

---

## Дополнительные инструменты

| Инструмент                    | Назначение                                                   |
| ----------------------------- | ------------------------------------------------------------ |
| [Phoenix BSL](phoenix-bsl.md) | Статический анализ кода, проверка на соответствие стандартам |
| [VS Code](vscode.md)          | Редактирование кода BSL, работа с Git, автодополнение        |
| [1С:EDT](edt.md)              | Полноценная IDE с поддержкой Git и рефакторинга              |

---

## Когда использовать

| Задача                                 | Инструмент                 |
| -------------------------------------- | -------------------------- |
| Редактирование метаданных конфигурации | Конфигуратор               |
| Работа с хранилищем 1С                 | Конфигуратор               |
| Проверка кода перед коммитом           | Конфигуратор + Phoenix BSL |
| Работа с Git (просмотр истории, ветки) | VS Code + расширения       |
| Все, что делаете в конфигураторе + Git | 1С:EDT + расширения        |

---

## Рекомендации

1. **Проверяйте код** с помощью Phoenix BSL перед каждым помещением в хранилище или коммитом
2. **Используйте VS Code** для быстрого редактирования BSL-файлов и работы с Git
3. **Изучите 1С:EDT**, если планируете полноценную работу с Git без хранилища 1С

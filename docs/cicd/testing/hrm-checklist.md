title: Чек-лист ЗУП
---

Чек-лист предназначен для функциональной проверки изменений в 1С:ЗУП и не зависит от конкретной реализации CI/CD контура.

## 1. Кадровый контур

1. Прием на работу.
2. Кадровый перевод.
3. Увольнение.
4. Отпуск (включая перенос/отзыв при наличии сценария).
5. Больничный лист.

## 2. Расчетный контур

1. Начисление зарплаты и взносов за период.
2. Перерасчет начислений.
3. Расчет налога на доходы физических лиц.
4. Выплаты между расчетными периодами (если применимо).
5. Выплата зарплаты.

## 3. Закрытие периода

1. Проверка выполнения регламентных операций периода.
2. Проверка отсутствия критических ошибок при закрытии месяца.
3. Проверка корректности итоговых сумм по контрольным отчетам.

## 4. Отчетность и печатные формы

1. Формирование ключевых расчетных отчетов.
2. Проверка печатных форм по измененным объектам.
3. Проверка пользовательских вариантов отчетов (при наличии).

## 5. Права и доступ

1. Проверка входа и основных сценариев под ключевыми ролями.
2. Проверка ограничений доступа к данным (если используется RLS).
3. Проверка отсутствия избыточных прав у прикладных ролей.

## 6. Интеграции и обмены

1. Проверка ключевых обменов с внешними системами (если применимо).
2. Проверка очередей/регламентных заданий, участвующих в обмене.
3. Проверка корректной обработки ошибок интеграции.

## 7. Нефункциональные проверки

1. Проверка времени выполнения критических операций на типовом объеме данных.
2. Проверка отсутствия блокирующих ошибок в журнале регистрации.
3. Проверка корректного завершения фоновых и регламентных заданий.

## 8. Критерии завершения проверки

1. Нет блокирующих дефектов по критическим сценариям.
2. Пройден полный обязательный контур чек-листа.
3. Зафиксированы результаты проверки и решение о готовности.

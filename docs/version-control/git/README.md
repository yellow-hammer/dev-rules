### Шаблон проекта

Для новых проектов используйте шаблон [vanessa-bootstrap](https://github.com/yellow-hammer/vanessa-bootstrap) — готовая структура репозитория для разработки на 1С с поддержкой:

- Конфигурации и расширений
- CI/CD пайплайнов
- Автоматического тестирования
- Генерации CHANGELOG
- Запуск задач из VS Code

---

### Когда использовать Git

- Разработка расширений конфигурации
- Разработка внешних обработок и отчетов
- Управление версиями скриптов и утилит
- Совместная работа над проектами
- Интеграция с CI/CD системами
